# 高级渲染技术

这个阶段主要学习 WebGL 的高级渲染技术，包括高级着色器编程、纹理技术和阴影渲染。

## 目录结构

### 01-shaders（高级着色器编程）
- 自定义着色器开发
- 多 Pass 渲染
- 后期处理效果
  - 泛光（Bloom）
  - 景深（DOF）
  - 运动模糊
  - SSAO

### 02-textures（高级纹理技术）
- 多重纹理和纹理数组
- 立方体贴图和环境映射
- 法线贴图和视差贴图
- PBR 材质系统实现

### 03-shadows（阴影技术）
- 阴影映射基础
- PCF 软阴影
- VSM（方差阴影映射）
- PCSS（百分比近似软阴影）

## 学习目标
1. 掌握高级着色器编程技术
2. 理解并实现各种现代渲染效果
3. 掌握高级纹理和阴影技术的实现原理

## 预计时间
2-3 周
